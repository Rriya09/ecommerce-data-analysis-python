# E-Commerce Sales Analysis using Python, Pandas, NumPy & Matplotlib

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an e-commerce dataset using Python. It focuses on data cleaning, sales analysis, customer segmentation, product recommendations, statistical analysis, and data visualization.

The project demonstrates the practical use of Pandas for data manipulation, NumPy for numerical computations, and Matplotlib for creating insightful visualizations.


## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib


## Dataset Information

The dataset contains the following columns:

* CustomerID
* Description (Product Name)
* Quantity
* UnitPrice

A new column named 'PurchaseAmount' is created using:

```
PurchaseAmount = Quantity × UnitPrice
```

## Data Cleaning Process

The following preprocessing steps were performed:

* Selected only the required columns.
* Removed leading and trailing spaces from product names.
* Converted product names to Title Case.
* Created the `PurchaseAmount` column.
* Identified and removed duplicate records.
* Checked missing values in all columns.
* Removed rows having missing `CustomerID` values.

## Statistical Analysis

The project calculates:

* Average purchase amount for each product.
* Standard deviation of purchase amounts.
* Total sales generated by each product.
* Top 5 best-selling products.
* Lowest 3 selling products.
* Total spending by every customer.
* Top 10 VIP customers based on spending.

## Customer Segmentation

Customers are segmented using quartiles of their total spending:

* Low Spending Customers (≤ 25%)
* Medium Spending Customers (25% - 50%)
* High Spending Customers (50% - 75%)
* VIP Customers (> 75%)

This helps identify valuable customers and spending patterns.

## Product Recommendation

A simple product recommendation system has been implemented.

For a selected customer:

* Products already purchased are identified.
* Remaining products are extracted using NumPy's `setdiff1d()`.
* The highest-selling products that the customer has not purchased are recommended.

## NumPy Performance Comparison

The project compares:

* NumPy's 'mean()' function
* Traditional Python loops

This demonstrates the performance advantage of vectorized operations provided by NumPy.
